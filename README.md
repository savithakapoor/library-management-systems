# Library Management System

A console-based Library Management System developed in C++ using Object-Oriented Programming and File Handling concepts.

## Features

- Add New Books
- Display All Books
- Search Books by Title or Author
- Issue Books
- Return Books
- Persistent Data Storage using File Handling

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Console Application

## Concepts Implemented

- Classes and Objects
- Encapsulation
- Functions
- File Handling using fstream
- Menu-Driven Programming

## Project Description

This project simulates a basic library management system where users can add books, search for books, issue books, and return books. The application stores book records permanently using file handling.

## How to Run

1. Open the project in any C++ IDE (Code::Blocks / Dev C++ / VS Code).
2. Compile the program.
3. Run the executable file.
4. Use the menu options to perform library operations.
